Laboratorio 12 - Funciones en JavaScript

Función: esPar()

Determina si un número ingresado es par o impar.

Función: áreaRectangulo()

Calcula el área de un rectángulo a partir de su base y altura.

Función: esMultiplo()

Verifica si el primer número ingresado es múltiplo del segundo.

Función: mayorDeTres()

Devuelve el mayor número entre tres valores ingresados.

Función: promedio()

Calcula el promedio de tres notas ingresadas.

Función: convertirAMayusc()

Convierte el texto ingresado a mayúsculas.

Función: calcularDescuento()

Calcula el precio final aplicando un porcentaje de descuento.

Función: saludo()

Devuelve un mensaje de saludo personalizado según el nombre ingresado.

Función: esPositivo()

Indica si el número ingresado es mayor que cero.

Función: componerTransformaciones()

Combina dos transformaciones de texto (mayúsculas y signo) en una sola función.

Función: procesarTexto()

Cuenta las palabras de un texto después de limpiar los espacios extra.

Función: operacionesMatemáticas()

Realiza las operaciones básicas (suma, resta, multiplicación y división) con dos números.

Función: crearContador()

Crea un contador que puede incrementarse o reiniciarse usando closures y funciones flecha.

Función: acumulador()

Crea un acumulador que inicia en un valor y va sumando nuevos valores cada vez que se invoca.

Función: generarSaludoPersonalizado()

Genera un saludo personalizado utilizando un closure que recuerda el nombre y la edad.

Función: media(...numeros)

Calcula el promedio de varios números utilizando el operador rest.

Función: mostrarDatos(nombre, edad, ...hobbies)

Muestra los datos personales y una lista de hobbies usando el operador rest.

Función: ejecutarOperacion(fn, x, y)

Ejecuta una operación aritmética pasada como función junto con dos valores numéricos.

Función: filtrarArreglo(arr, callback)

Filtra un arreglo de números según la condición elegida.

Función: descargarArchivo(url, callback)

Simula la descarga de un archivo y muestra un mensaje cuando finaliza.

Función: potencia(base, exponente)

Calcula la potencia de un número usando solo recursividad (sin Math.pow ni **).

Función: crearSecuencia(inicio, paso)

Genera una secuencia aritmética que incrementa en cada llamada sin usar arreglos ni objetos.
